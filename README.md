# homeexpenseapp
## 🚀 Features

- ➕ Add daily income and expenses
- 🗂️ Categorize expenses (Food, Travel, Bills, etc.)
- 📊 View total balance, income, and expenses
- 🧾 Transaction history tracking
- ✏️ Edit and delete transactions
